# rabbit-utils
### HOW TO USE
##### Maven
- Add the JitPack repository
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
- Add the dependency
```xml
<dependency>
    <groupId>com.github.ProgrammerRabbit</groupId>
    <artifactId>rabbit-utils</artifactId>
    <version>-df59abe3d8-1</version>
</dependency>
```
##### Jar
```
rabbit-utils--df59abe3d8-1.jar
```
